# 输出风格模板

Phase 2 让用户选择风格时，从以下 5 种中展示 2-3 个让用户挑。AI 展示样式预览图或描述，用户选一个后按该规范生成 PPT。

---

## 风格 1：TED 演讲风 🎤

**调性**：克制、有力量、视觉极度简洁，文字是配角，观点是主角。

**规范**：
- 配色：深底浅字（背景 `#000000` 或 `#1A1A2E`，文字 `#FFFFFF`）；强调色只用一个（推荐 `#FF6B35` 或品牌色）
- 字体：无衬线大字体，标题 **48-60px**，正文 **24-28px**，行距 1.5x
- 每页：**不超过 1 个观点**，标题即观点，不写废话
- 图片：全屏背景图，深色遮罩，白字叠加（hero image 风格）
- 过渡：极简，`none` 或 `fade`

**适合**：公开分享、激励型演讲、观点输出强的场景

---

## 风格 2：McKinsey 汇报风 💼

**调性**：专业、数据驱动、结论先行，每一页都在说服。

**规范**：
- 配色：深蓝白灰（背景 `#FFFFFF` 或 `#F8F9FA`，标题区 `#003366`，文字 `#1A1A2E`）
- 字体：衬线标题 + 无衬线正文（标题 `Georgia/Noto Serif SC`，正文 `Noto Sans SC`）
- 每页：**结论在顶部**，用金字塔结构（结论 → 3 个支撑点）
- 数据：能用图表不用表格，能用表格不用文字
- 过渡：`convex`，专业感

**适合**：领导汇报、客户提案、需要建立权威感的场景

---

## 风格 3：Google 设计风 🎨

**调性**：年轻、多彩、圆角、图标化，像 Material Design 活过来了。

**规范**：
- 配色：多彩但统一（主色 `+` 系列：`#4285F4`蓝 / `#EA4335`红 / `#FBBC04`黄 / `#34A853`绿）
- 字体：Google Sans 风格（用 `Noto Sans SC`，圆润感）
- 每页：图标 + 短句，避免长段落；圆角卡片（border-radius 16px+）
- 留白：大量留白，不要把页面填满
- 过渡：`zoom` 或 `concave`，有活力

**适合**：内部培训、年轻团队分享、产品介绍

---

## 风格 4：极简黑白风 ⚪

**调性**：少即是多，只留最重要的，其他全部砍掉。

**规范**：
- 配色：黑白灰（背景 `#FFFFFF`，文字 `#000000`，灰 `#888888`）
- 字体：纤细无衬线（标题细体，正文常规）
- 每页：**1 个元素**（一句话 / 一张图 / 一组数据），不叠加
- 对齐：严格左对齐或居中，不混用
- 过渡：`none`，切换就是翻页

**适合**：设计分享、思考类话题、需要"留白让观众思考"的场景

---

## 风格 5：活泼培训风（默认） 🔥

**调性**：热情、互动强、有游戏感，适合培训场景。

**规范**：
- 配色：橙色系（主色 `#FF6B35`，背景 `#FFF7F0`，强调 `#FFB800`）
- 字体：阿里普惠体（Noto Sans SC），圆润友好
- 每页：有图标、有互动元素（翻转卡/投票），信息密度中等
- 动画：适度使用，引导注意力但不抢夺
- 过渡：`convex`，轻松感

**适合**：企业培训、内部分享、互动多的场景（**本 Skill 默认风格**）

---

## 如何选风格（AI 决策树）

```
问：audience 是谁 + 分享目的？
│
├── 公开演讲/激励 → TED 演讲风
├── 领导/客户/要说服 → McKinsey 汇报风
├── 年轻团队/内部培训 → 活泼培训风（默认）
├── 设计/创意/产品 → 极简黑白风 或 Google 设计风
└── 不确定 → 问用户选，展示 2 个风格描述让用户挑
```
