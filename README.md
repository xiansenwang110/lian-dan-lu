# 炼丹炉 (lian-dan-lu)

> 将课程录音/文字记录/文档材料**炼**成结构化分享材料并制成 PPT 的完整工作流 Skill。

## 触发场景

用户听完一门课/看完一份材料，需要产出分享内容（PPT/网页/文章等），或说：
- 帮我炼一炉
- 做成分享材料
- 做成 PPT
- 把这门课整理成分享

## 核心流程

1. 输入材料 - 录音/文字记录/文档
2. 提炼核心观点 - AI 分析结构化
3. 生成 Markdown 分享稿 - 逻辑清晰、有节奏
4. 转化为离线 HTML PPT - 自写 JS 翻页系统，零外部依赖，微信可直接打开
5. 迭代修改 - 用户反馈 -> 调整内容
6. 交付 - 最终 PPT 输出

## 适用场景

培训课程、讲座录音、读书笔记、会议记录，以及任何需要「学习 -> 提炼 -> 输出」的场景。

## 输出形态

默认输出：离线 HTML PPT（自写 JS，零外部依赖，微信可直接打开）
